# 🩸 Blood on the Clocktower AI Copilot | 《血染钟楼》玩家智能记录与逻辑辅助系统

这是一个专为《血染钟楼》**玩家**打造的纯前端单页面记录与逻辑推理辅助工具。

系统采用**暗色磨砂玻璃（Premium Dark-Glassmorphism）**视觉设计，集成全球主流 AI 接口，旨在帮助玩家在对局中高效记录玩家发言、厘清座椅视角、分析角色冲突，并借助前沿 AI 大模型盘清世界线，提供具有深度的逻辑论证与战术发言建议。

---

## 🌟 核心功能

### 1. 🔄 第一人称动态座椅旋转视角 (First-Person Seating)
* **主视角锚定**：输入您自身的座位号，圆桌图会自动旋转，将您的座位锁定在**正下方（6点钟方向）**。
* **自适应排布**：其余玩家座位根据您的主视角自动呈顺时针重新计算位置，提供最直观的对面对局视角。
* **状态指向线旋转**：玩家之间的技能指向线、中毒/醉酒状态会随圆桌动态旋转，保证视觉逻辑清晰一致。

### 2. 🧠 多通道前沿 AI 逻辑辅助 (AI Logic Companion)
玩家可以将当前的对局信息（存活状态、角色宣称、逻辑冲突、当前天数）整理后发送给 AI，系统将从三个维度提供逻辑支持：
* **局势即时分析**：自动梳理场上已知的逻辑矛盾与角色宣称冲突。
* **世界线推演**：根据不同狼人/恶魔在野的假设，推演多种可能的局势走向。
* **战术发言建议**：针对当前轮次，为玩家提供发言策略、投票倾向及排坑建议。
* *安全机制：直连全球 9 大主流 AI 平台（包含 Gemini 3.5、GPT-5.5、Claude 4.8、DeepSeek V4 等），不经过任何第三方中转服务器，保障数据隐私。*

### 3. 📖 四大主流剧本对照表与规则约束 (Multi-Script Integration)
内置四个热门板子的完整角色表，切换剧本时右侧的技能表和 AI 提示词约束会自动实时更新：
1. **《无上杀戮》 (Supreme Slaughter)**
2. **《暗流涌动》 (Trouble Brewing)**
3. **《梦殒春宵》 (Sects & Violets)**
4. **《瓦釜雷鸣》 (Laissez-Faire)**
* **时机精确审计**：严格审计了所有角色技能的发动时机（首夜不发动的动作均清晰标注了 `每个夜晚*`）。
* **剧本细节差异**：完美区分了同名角色在不同剧本中的细微差异（例如 *气球驾驶员*，在《无上杀戮》中为 `[+0或+1外来者]`，而在《瓦釜雷鸣》中则是固定的 `[+1外来者]`）。

### 4. 🛡️ 高可靠性本地会话保护与离线状态管理
* **状态实时保存**：玩家修改的每个座位状态、生死存亡、玩家小备注、草稿箱和 AI 历史分析结果，都会**实时静默保存至浏览器的 `localStorage` 中**。即便发生网页意外刷新或浏览器闪退，启动时即可秒级恢复现场。
* **离线安全队列**：实时监测浏览器在线状态。断网时 API 请求会被安全加入本地队列并发出友好提示，确保珍贵的对局记录 0% 丢失。

## 🚀 快速上手指南

本系统为**纯前端单页面应用（SPA）**，无需任何繁琐的后端环境配置，开箱即用。

### 方法 A：直接双击运行
1. 将项目下载或克隆至本地。
2. 双击项目根目录下的 `index.html` 即可在任意现代浏览器中运行。

### 方法 B：通过本地 Web 服务器运行（推荐）
如果您电脑中安装了 Python，可在项目根目录下通过终端启动简易服务器：
```bash
python -m http.server 8000
```
启动后在浏览器中访问 `http://localhost:8000` 即可。

## 🔒 隐私与安全承诺
* **数据零上传**：应用不包含任何后端数据库，您的所有 API Key、对局小记、玩家信息均仅存储在您本地浏览器的 `localStorage` 中。
* **密钥交叉防泄露**：在切换不同厂商模型时，若检测到前缀不匹配（例如在 Google 接口输入 `sk-` 开头的密钥），系统会自动清空并提醒，杜绝密钥无意向第三方中转站泄露的风险。
