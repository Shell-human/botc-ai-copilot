# 🩸 Blood on the Clocktower AI Copilot | 血染钟楼玩家好帮手

这是一个专门为《血染钟楼》**玩家**打造的纯前端单页面对局记录与逻辑辅助工具。

打本盘逻辑时，是不是经常觉得座位对不上、记不住谁跳了什么角色、或者是到了关键轮次脑子转不过来？这个小工具就是为了解决这些痛点而生的！它能帮你轻松记录场上局势，还能一键接入 AI 帮你盘世界线、提供战术点子。

---

## 🌟 核心功能

### 1. 🔄 座椅图“第一人称”旋转视角
* **以“我”为中心**：输入你自己的座位号，圆桌图会自动旋转，把你的座位雷打不动地放在**正下方（6点钟方向）**。
* **对号入座**：其余玩家按顺时针自动排布，从你的真实视角看过去谁在左边、谁在右边一目了然，再也不用歪着脖子对人名了。
* **状态连线**：玩家之间的技能指向线、中毒/醉酒状态会跟着圆桌一起旋转，视觉逻辑不打架。

### 2. 🧠 AI 局势盘道与战术支招 (支持 2026 最新大模型)
你可以把当前的对局信息（谁死了、谁穿了什么衣服、谁站边谁、当前是第几天）整理好，一键发给 AI。AI 会从三个维度帮你出谋划策：
* **局势即时分析**：盘出场上的疑点和逻辑冲突。
* **世界线推演**：帮你假设“如果是A狼/如果是B狼”会有什么样不同的故事走向。
* **战术小贴士**：当前轮次你该怎么发言、怎么投票、如何排坑。
* *支持直连全球 9 大主流 AI 平台（包含 Gemini 3.5、GPT-5.5、Claude 4.8、DeepSeek V4 等），不经过任何第三方服务器，安全省心。*

### 3. 📖 四大主流剧本内置与细节校对
工具内置了四个热门板子的完整角色表，切换板子时右侧的技能表和 AI 提示词会自动更新：
1. **《无上杀戮》 (Supreme Slaughter)**
2. **《暗流涌动》 (Trouble Brewing)**
3. **《梦殒春宵》 (Sects & Violets)**
4. **《瓦釜雷鸣》 (Laissez-Faire)**
* **技能细节校对**：严格审计了所有角色技能的发动时机（首夜不发动的动作全部清晰标注了 `每个夜晚*`）。
* **跨板子差异**：完美区分了同名角色在不同板子里的细微差别（比如 *气球驾驶员*，在《无上杀戮》里是 `[+0或+1外来者]`，而在《瓦釜雷鸣》里则是固定的 `[+1外来者]`）。

### 4. 🛡️ 防手残自动保存与离线支持
* **闪退不用慌**：你修改的每个座位、改动的生死状态、随手写的玩家小笔记、草稿箱和 AI 的回复，都会**在后台实时自动保存到你浏览器的 `localStorage` 中**。哪怕不小心刷新了网页或者浏览器闪退，一打开立刻恢复原样。
* **离线也能记**：实时检测网络状态。断网时点发送不会报错丢失数据，工具会把你的发言存入本地，提示你等联网后一键重新发送。

## 🚀 怎么用？

本工具是**纯前端单页面**，不需要安装任何繁琐的运行环境，下载即用。

### 方法 A：直接双击（最简单）
1. 把代码克隆或下载到本地。
2. 在文件夹里找到 `index.html`，双击用任意浏览器打开即可。

### 方法 B：本地起个简易服务器（推荐，体验更丝滑）
如果你电脑里装了 Python，在项目文件夹里开个终端运行：
```bash
python -m http.server 8000
```
然后在浏览器里输入 `http://localhost:8000` 就可以玩啦。

## 🔒 隐私与安全
* **不存服务器**：本工具没有任何后台服务器。你的所有 API Key、游戏备注、对局记录全部只存在你自己的浏览器缓存里，绝不上传，防泄露。
* **防呆密钥清理**：切换不同厂商的大模型时，如果检测到密钥前缀不匹配（比如在 Google 选框里留着 ChatGPT 的 `sk-` 密钥），工具会自动帮你清空，防止把 Key 误传给其他接口。
