# 🩸 Blood on the Clocktower AI Copilot | 《血染钟楼》智能裁判辅助决策系统

这是一个专为《血染钟楼》说书人（Storyteller / 裁判）打造的次世代 AI 辅助决策系统。采用**暗色霓虹磨砂玻璃（Premium Dark-Glassmorphism）**美学设计，深度集成全球九大主流 AI 厂商的 2026 前沿大模型，提供第一人称动态座椅布局、多剧本规则实时审计、离线防崩溃自动恢复等功能，助力说书人轻松驾驭各种复杂对局。

---

## ✨ 核心特色功能

### 1. 🔄 第一人称动态座椅旋转视角 (First-Person Seating Rotation)
* **主视角锚定**：将说书人或您的座位（“我”）始终数学锁定在圆桌正下方（6点钟方向）。
* **动态环形重排**：其余玩家座位根据您的视角自动呈顺时针动态排布，完美还原面对面实战沙盘视角。
* **自适应保护线**：守护与中毒/醉酒状态等 SVG 关联指向线随圆桌动态旋转，保证视觉逻辑 100% 清晰。

### 2. 🤖 九大前沿 AI 厂商与 2026 旗舰模型集成
深度适配最新的 API 调用规范，支持预填各厂商 Base URL，涵盖以下顶级模型：
* **Google Gemini**: `gemini-3.5-flash`（2026全新核心推理模型）、`gemini-3.1-pro-preview`（2M超长上下文）
* **OpenAI ChatGPT**: `gpt-5.5`（2026旗舰核心）、`gpt-5.4`、`gpt-5.4-mini`
* **Anthropic Claude**: `claude-opus-4-8`（前沿推理旗舰）、`claude-sonnet-4-6`
* **DeepSeek**: `deepseek-v4-pro`（深度思考 MoE 旗舰）、`deepseek-v4-flash`
* 以及 **阿里通义千问 (Qwen3.7-Max)**、**智谱清言 (GLM-5.1)**、**字节跳动 (Doubao Seed 2.0 Pro)**、**月之暗面 (Kimi K2.6)**、**百度文心 (ERNIE 5.1)**。
* *安全防护：提供跨厂商切换时密钥自动防泄露/清空逻辑。*

### 3. 📖 四大主流剧本对照表与规则实时审计
内置四大热门剧本的完整角色库与技能说明，切换剧本时自动更新界面、生成专属 AI Prompt 提示词约束：
1. **《无上杀戮》 (Supreme Slaughter)**
2. **《暗流涌动》 (Trouble Brewing)**
3. **《梦殒春宵》 (Sects & Violets)**
4. **《瓦釜雷鸣》 (Laissez-Faire)**
* **规则校对**：严格校对了 70+ 角色的发动时机（例如 Night 2+ 动作统一标记为 `每个夜晚*`）。
* **同名差异化**：完美处理了跨剧本同名角色的细节差异（例如 *气球驾驶员* 在《无上杀戮》中为 `[+0或+1外来者]`，而在《瓦釜雷鸣》中为 `[+1外来者]`）。

### 4. 🛡️ 离线防崩溃与实时自动保存系统
* **无缝防崩溃**：页面操作、玩家存活状态、警报状态、草稿输入、AI 分析结果等在每次变更时均会**实时静默保存至 `localStorage`**。页面意外刷新或浏览器崩溃时，启动即可秒级自动恢复。
* **断网状态队列**：实时监测浏览器在线/离线状态。当处于离线状态时，API 请求会安全加入本地队列并发出友好提示，确保珍贵的对局记录 0% 丢失。

## 🚀 快速上手指南

本系统为**纯前端单页面应用（SPA）**，无需任何复杂的后端安装或环境配置，开箱即用！

### 方法 A：直接双击运行（最简方式）
1. 将本项目克隆或下载到本地。
2. 双击文件夹中的 `index.html` 即可在任意现代浏览器中运行。

### 方法 B：通过本地 Web 服务器运行（推荐，防跨域及性能更佳）
如果您安装了 Python，可以在项目根目录下启动一个极简服务器：
```bash
# 启动本地服务器
python -m http.server 8000
```
然后在浏览器中访问：`http://localhost:8000` 即可。

* **结构层 (Structure)**: 语义化 HTML5，利用 CSS 变量及现代布局技术。
* **表现层 (Styling)**: 原生 Vanilla CSS，融入高阶玻璃拟态（Glassmorphism）、微卡片高亮、霓虹呼吸灯效与平滑运动过渡（Transitions）。

## 🔒 隐私与安全承诺
* **零后台记录**：本应用不设任何云端数据库。您的所有 API Key、游戏状态、玩家备注均仅保存在您本地浏览器的 `localStorage` 中。
* **接口安全**：切换不同大模型时，若检测到非对应格式密钥（例如在 Google 接口输入 `sk-`），系统会自动拦截并提醒清空，杜绝密钥无意泄露风险。
