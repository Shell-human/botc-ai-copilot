# 🩸 血染钟楼 AI 战术推理逻辑核心知识库 (Deduction Axioms)

本知识库梳理了《血染钟楼》对局推演中**具有绝对物理确定性、边界极其严苛的钢性逻辑与推演公理**。无论是 AI 战术副驾驶的底层逻辑大脑（Constraint Solver），还是高阶人类玩家在复盘与实战中的算力求解，均以此为核心基石。

---

## 🛑 核心推理公理 (Deduction Axioms)

### 1. 伪装排他律 (Demon Bluffs Exclusivity)
*   **公理描述**: 说书人首夜给恶魔分配的 3 个 Bluffs 角色在当前魔典真实在场玩家中 **[0 存在]**。
*   **逻辑推论**: 
    1.  场上任何起跳宣称 Bluffs 角色的玩家，其宣称身份**必为伪报**。
    2.  其真实背景必定为以下五类之一：邪恶穿衣伪装、被洗脑师强行疯狂、畸形秀自保、酒鬼/疯子，或好人高级挡刀战术。
*   **邪恶利用**: 邪恶同伴必须在得知 Bluffs 后快速对齐“谁穿哪件皮肤”，确保无人撞衫，并主动穿 Bluffs 自做高身份。

### 2. 邪恶共知律 (Evil Mutual Recognition & PG Rule)
*   **公理描述**: 邪恶阵营默认首夜闭眼相认并掌握彼此的位置（爪牙知恶魔，恶魔知爪牙）。
*   **罂粟干预**: 若邪恶玩家在开局首夜**未相认**，判定 **[罂粟种植者] 百分之百在场**。
*   **逻辑约束**: 在罂粟种植者死亡当夜的说书人夜间重组结算前，邪恶同伴绝对不可私下对暗号认亲，以防撞刀自爆。

### 3. 座位收缩律 (Seating Contraction)
*   **公理描述**: 夹在存活玩家中间的已死玩家，在判定邻座关系（共情者验人、茶艺师保护、诺达鲺中毒）时**直接被物理消除**，两侧存活玩家向内并拢接轨，建立新的“存活邻座”邻接线。
*   **推理应用**: 共情者等空间位置玩家，在死人前后的信息变动（如 1 号死后共情者信息从 1 变为 0），可以直接通过**算术差值求解**锁定已死或两侧新存活玩家的阵营属性。

### 4. 涡流反转律 (Vortox Absolute Lie Constraint)
*   **公理描述**: 若恶魔为 **[涡流]**，所有镇民（Townsfolk）的技能信息 **[100% 必须为假]**。
*   **逻辑推论**: 
    1.  场上若有任何一名镇民被绝对证实得到了真话信息，当前恶魔**百分之百不是涡流**。
    2.  若恶魔为涡流，将所有镇民提供的信息进行 **100% 的完全逻辑取反**，即是GrMoire的真实魔典状态。
    3.  外来者信息不受此限；白天无人被处决时邪恶直接获胜，因此好人必须**处决续命**。

### 5. 角色唯一律 (Good Character Uniqueness)
*   **公理描述**: 初始分配魔典中，每个善良城镇角色（Townsfolk）在场上**至多有 1 个**。
*   **逻辑推论**: 两人宣称同一善良身份（撞衫），且排除洗脑师疯狂、麻脸巫婆深夜转化、或酒鬼干扰时，**此二人必有一人为邪恶或酒鬼/疯子**。

### 6. 疯狂雷击律 (Madness Thunderstruck)
*   **公理描述**: 畸形秀演员疯狂证明自己是外来者，或洗脑受害者白天未能疯狂扮演被洗脑善良角色时，会面临说书人当场的**非提名直接惩罚处决**。
*   **推理应用**: 白天在没有发起任何提名的情况下突然发生玩家雷击猝死，**优先定位洗脑师/畸形秀**的在场及疯狂目标座位。

### 7. 恶魔承袭律 (Demon Lineage Succession)
*   **公理描述**: 恶魔的传承有严苛的人数与角色触发规则。
*   **逻辑推论**: 
    1.  **小恶魔承袭**: 场上活人 >= 5 人时小恶魔选择夜间自刀，**必然**有一名爪牙玩家当夜晋升转变为新小恶魔继承王位。
    2.  **红唇女郎承袭**: 场上活人 >= 5 人（不含旅行者）时恶魔死亡，**红唇女郎当夜必然承袭恶魔位置**。

## 📊 推理逻辑判定速查表 (Deduction Quick Reference)

| 触发事件 (Event) | 排除干扰后的硬核推导 (Deduction Laws) |
| :--- | :--- |
| **宣称起跳恶魔给我的 3 个 Bluffs 角色** | 该玩家 100% 身份有诈（邪恶穿衣 / 洗脑疯狂 / 畸形秀 / 酒鬼 / 疯子 / 好人挡刀）。 |
| **首夜说书人没让我（狼）认队友** | 场上 100% 存在【罂粟种植者】。未死前，严禁对暗号认亲。 |
| **死人前后共情者/茶艺师邻座状态发生重组** | 通过重组前后的数学数值改变，直接反推新旧邻座的善良/邪恶本源。 |
| **任何镇民玩家拿到了绝对证实正确的信息** | 当前恶魔 100% 绝非【涡流】。 |
| **白天在没有任何提名情况下发生离奇猝死** | 场上 100% 存在【洗脑师】或【畸形秀演员】违背疯狂引发说书人惩罚处决。 |
| **贞洁者被其他玩家提名后没有触发闪劈处决** | 提名者 100% 不是健康的好人镇民（可能是狼、外来者、或是醉酒中毒的镇民）。 |
| **恶魔夜间自刀且场上当晚没有死两个活人** | 邪恶阵营当晚必然有一人转变为【新小恶魔】继承皇位。 |
